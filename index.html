SUI-Card React component

An open source React card component

Description

The SUI-Card component features a generic div tag including two more nested containers one on top of the other.

Top and bottom containers may fit any content provided to Sui-Card. Use the following classes to layout styling:

  • sui-Card-topComponent:

    This is the top placeholder where you can fit any custom component: from a simple picture to a full featured slider.

    In the example below an anchor element wrapping an image is stored in constant 'top' and passed to SUI-Card component as 'top-content'.

  • sui-Card-bottomComponent:

    This is the bottom placeholder. Pass a custom component or insert your static HTML code as shown in example.

Sub-components data structure

        
    // The content of constant 'top' will be inserted
    // inside sui-Card-topComponent sub-component
    const top = (
        <a href='http://bit.ly/1jt4zRp'>
          <img src='http://bit.ly/1FzqHDX' />
        </a>
    );

    // The content of constant 'bottom' will be inserted
    // inside sui-Card-bottomComponent sub-component
    const bottom = (
      <div>
        <h2>Fishermen at work</h2>
        <p>Lorem ipsum dolor sit amet,
        consectetur adipisicing elit.</p>
        <a href='http://google.com'>Contact</a>
      </div>
    );
